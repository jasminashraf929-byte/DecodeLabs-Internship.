**DecodeLabs Full Stack Internship | Batch 2026**

## Overview

A fully responsive frontend interface for a portfolio web application built using only HTML5, CSS3, and Vanilla JavaScript — no frameworks.

---

## Live Preview

Open `index.html` directly in any browser. No server required.

---

## Features

- Responsive layout across mobile (< 768px), tablet (768px), desktop (1024px+)
- Semantic HTML5 landmarks — `header`, `nav`, `main`, `article`, `footer`
- CSS Grid for macro layouts + Flexbox for components
- Mobile-first design with hamburger navigation
- Animated stat counters using IntersectionObserver
- Contact form with client-side validation and toast notifications
- Scroll-to-top button
- WCAG accessibility — skip link, ARIA labels, keyboard navigation
- 2025 design palette: Mocha Mousse `#A5856E`, Ethereal Blue `#A0D4E0`, Moonlit Grey `#F2F0EA`
- Typography: Montserrat (headlines) + Roboto (body)
